# Addressbook
A menu-driven address book application developed in C, designed to efficiently manage contact information with features such as adding, searching, updating, and deleting records. The project demonstrates structured programming and file handling concepts.

## Features
- Add new contact
- Search contact by name,number or email
- Edit existing contact details
- Delete contact
- Display all contacts

## Technologies Used
 - File operations 
 - Pointers 
 - Structures
 - Functions 
 - Makefiles 

 ## How to Run
 1. Compile the program
    gcc *.c -o addressbook (To compile the the files at the same time)
 2. Run the code
    ./addressbook
